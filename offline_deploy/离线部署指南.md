# 变电站图像监控运维平台 — 内网离线部署指南

> 目标：在不能联网的 Windows 电脑上部署本平台

## 一、准备清单（在联网电脑上操作）

### 1.1 需要下载的文件

| 文件 | 下载地址 | 用途 |
|------|----------|------|
| Python 3.11 安装包 | https://www.python.org/ftp/python/3.11.9/python-3.11.9-amd64.exe | Python运行环境 |
| PostgreSQL 16 安装包 | https://get.enterprisedb.com/postgresql/postgresql-16.8-1-windows-x64.exe | 数据库（可选，也可用SQLite） |
| 项目代码 | `E:\项目\变电站图像监控运维平台` 整个文件夹 | 平台代码 |
| 依赖包 | `offline_deploy\packages\` 文件夹 | Python依赖（已下载好） |

### 1.2 拷贝到U盘

把以下内容拷到U盘：
```
U盘/
├── python-3.11.9-amd64.exe          ← Python安装包
├── postgresql-16.8-1-windows-x64.exe ← 数据库安装包（可选）
├── 变电站图像监控运维平台/             ← 整个项目文件夹
└── 离线部署指南.md                    ← 本文件
```

## 二、内网电脑安装步骤

### 2.1 安装 Python

1. 双击 `python-3.11.9-amd64.exe`
2. **重要：勾选 "Add Python to PATH"**
3. 点击 Install Now
4. 安装完成后，打开 CMD 验证：
   ```
   python --version
   pip --version
   ```

### 2.2 安装 Python 依赖包

打开 CMD，进入项目目录：
```
cd /d X:\变电站图像监控运维平台
pip install --no-index --find-links=offline_deploy\packages -r requirements.txt
```

验证安装：
```
python -c "import flask; print('Flask', flask.__version__)"
```

### 2.3 安装 PostgreSQL（推荐，正式使用）

1. 双击 `postgresql-16.8-1-windows-x64.exe`
2. 安装过程中设置：
   - 安装目录：默认 `C:\Program Files\PostgreSQL\16`
   - 数据目录：默认
   - **超级用户密码**：设一个你记得住的密码（如 `Txjk@1234`）
   - 端口：**5432**（默认）
   - 区域：Default locale
3. 安装完成后验证：
   ```
   "C:\Program Files\PostgreSQL\16\bin\psql" -U postgres -c "SELECT version();"
   ```

创建数据库和用户：
```
"C:\Program Files\PostgreSQL\16\bin\psql" -U postgres
```
在 psql 里执行：
```sql
CREATE USER station_monitor WITH PASSWORD 'Txjk@1234';
CREATE DATABASE station_monitor OWNER station_monitor;
\q
```

### 2.4 配置环境变量

编辑项目目录下的 `.env` 文件：
```
DATABASE_URL=postgresql://station_monitor:Txjk@1234@127.0.0.1:5432/station_monitor
SECRET_KEY=随便写一串长字符比如abc123xyz789
PHOTO_ROOT_PATH=内网电脑上照片目录的路径
APP_DATA_DIR=内网电脑上数据目录的路径
API_TOKEN=随便写一个密钥比如my_secret_token_123
FLASK_DEBUG=False
```

如果暂时不想装 PostgreSQL，可以用 SQLite（零配置）：
把 `.env` 里的 `DATABASE_URL` 那一行删掉或留空就行。

### 2.5 初始化数据库

```
cd /d X:\变电站图像监控运维平台
python init_db.py
python init_admin.py --username admin --password 你的管理密码
```

### 2.6 启动平台

**方式A：命令行启动**
```
python app.py
```
看到 `Running on http://0.0.0.0:5000` 就成功了。

**方式B：双击启动**
- 双击 `start.bat` — 普通启动（有命令行窗口）
- 双击 `start_hidden.bat` — 后台启动（无窗口）
- 双击 `control_panel.pyw` — 图形控制面板

### 2.7 访问平台

在本机浏览器打开：
```
http://127.0.0.1:5000
```

内网其他电脑访问：
```
http://内网电脑IP:5000
```

如果其他电脑访问不了，检查 Windows 防火墙：
```
netsh advfirewall firewall add rule name="StationMonitor" dir=in action=allow protocol=TCP localport=5000
```

## 三、设置为开机自启动（可选）

### 方法：用 Windows 任务计划程序

1. 打开"任务计划程序"（taskschd.msc）
2. 创建基本任务
   - 名称：变电站图像监控运维平台
   - 触发器：计算机启动时
   - 操作：启动程序
   - 程序：`python.exe`
   - 参数：`app.py`
   - 起始于：`X:\变电站图像监控运维平台`
3. 勾选"不管用户是否登录都要运行"

---

## 四、仅用 SQLite 的最小部署（不装 PostgreSQL）

如果只是单机试用，不需要装 PostgreSQL：

1. 装好 Python
2. 装好依赖包
3. 编辑 `.env`，确保 `DATABASE_URL` 为空或删除该行
4. `python init_db.py`
5. `python init_admin.py --username admin --password 你的密码`
6. `python app.py`

SQLite 零配置，数据存在项目目录下的 `station_monitor.db` 文件中。

## 五、常见问题

### Q: pip install 报错找不到包
确保 `--find-links` 路径正确指向 `offline_deploy\packages`

### Q: 其他电脑访问不了
1. 确认内网电脑IP（CMD 里 `ipconfig`）
2. 确认防火墙放行了 5000 端口
3. 确认 `app.py` 监听的是 `0.0.0.0` 而不是 `127.0.0.1`

### Q: PostgreSQL 连接失败
1. 确认 PostgreSQL 服务已启动（services.msc 里查看）
2. 确认密码、端口、数据库名正确
3. 确认 `.env` 里的 `DATABASE_URL` 格式正确

### Q: 想迁移已有数据
把当前电脑的数据库导出：
```
pg_dump -U station_monitor station_monitor > backup.sql
```
在内网电脑导入：
```
psql -U station_monitor station_monitor < backup.sql
```
SQLite 直接拷贝 `station_monitor.db` 文件即可。
